Our Mission

Book Finder was created to help users easily discover books by title or author using real-time data. Whether you're researching for school or just looking for your next great read, our goal is to make book searching fast and enjoyable.

Technologies Used

  • HTML, CSS, JavaScript
  • Open Library API for book data
  • Chart.js for author frequency visualization
  • LocalStorage for saving favorites

Credits

This project was built as part of a web development course to demonstrate front-end design and API integration.
